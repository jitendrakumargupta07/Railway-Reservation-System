# Railway Reservation System

A full stack Railway Reservation System
This project allows users to **search trains, check seat availability, book or cancel tickets, and manage user accounts**. It integrates both **structured and non-structured data storage**, providing a complete railway booking experience.

---

## **Technologies Used**

### Frontend
- ReactJS
- TailwindCSS
- HTML & JavaScript
- Responsive design

### Backend
- NodeJS & ExpressJS
- REST APIs for train search, booking, cancellation, and authentication
- JSON Web Tokens (JWT) for secure login

### Database
- **MySQL** – structured data for trains, seats, and bookings
- **MongoDB** – logs, user activity, and dynamic non-relational data

---

## **Features**
- User authentication (Login/Register)
- Search trains by source and destination
- Check seat availability
- Book or cancel tickets
- Dashboard to manage bookings
- Clean and responsive user interface
